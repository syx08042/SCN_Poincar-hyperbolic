# SCN_Poincar-hyperbolic

Data description:
The dataset we are currently working with can be downloaded for free from the website https://github.com/JohnAbel/scn-resynchronization-data-2016
(J.H. Abel, K. Meeker, D. Granados-Fuentes, P.C. St. John, T.J. Wang, B.B. Bales, F.J. Doyle, E.D. Herzog, & L.R. Petzold, Functional network inference of the suprachiasmatic nucleus, Proc. Natl. Acad. Sci. U.S.A. 113 (16) 4512-4517, https://doi.org/10.1073/pnas.1521178113 (2016).)

Hyperbolic embedding method:
The hyperbolic embedding process is through Mercator from https://github.com/networkgeometry/mercator
(Guillermo García-Pérez et al 2019 New J. Phys. 21 123033)

Simulation of circadian rhythms:
The core code is in 'Simulate_Circadian.m'
